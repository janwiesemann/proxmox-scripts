# proxmox-scripts

A collection of automations I'm using in my Homelab.

## How to use

1. install git on your proxmox host
    ```
    apt install -y git
    ```
2. clone this repo
    ```
    git clone https://github.com/janwiesemann/proxmox-scripts.git
    ````

3. enter directory
    ```
    cd proxmox-scripts
    ````

4. execute scripts

    1. using bash
        ```
        bash script.sh
        ```
    
    2. or

        ```
        chmod +x script.sh
        ,/script.sh
        ```
